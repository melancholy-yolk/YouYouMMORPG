# YouYouMMORPG
a personal unity game made by unity and sql-server
both client and server project are coded by myself
工程结构：

服务器文件夹：
  1.账号服务器 http服务器 asp.net
  2.游戏服务器 c#+mssql

客户端文件夹：
  UI框架：基于MVC
  网络模块：http与TCP
  自定义proto：内存流自定义proto
  本地表格配置管理模块
  assetbundle一键打包工具
  资源打包、更新与下载
  角色模块：有限状态机、AI、a星寻路、战斗模块
  场景管理模块
  状态同步
